1. Getting Started

IPC COURT Welcome Page

  1. IPC COURT Installation - This is probably the simplest way for most users.
    1. Upload the source folder via ftp or file manager
    2. Create a database yourdbname.
    3. Using the link http://yoursitename.com/installer/index run the installer
    4. Fill out your database details in the installer and save your changes.
    5. A form for filling out your site details, smtp details will appear, enter your details and save your changes.
    6. Now you will get your site url, admin url, admin password and username

  2. Admin Login Details
    1. Login using the following url with username and password admin, admin into the IPC COURT admin panel.
    2. Username:admin
    3. Password:admin
    4. URL:http://ipccourt.co.in/admin
